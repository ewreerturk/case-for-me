You can access the Flask application using NodePort service.

To access:
1. Get the NodePort:
   NODE_PORT=$(kubectl get svc {{ include "flask-app.fullname" . }} -o jsonpath="{.spec.ports[0].nodePort}")

2. Get the IP address of any node in your cluster:
   NODE_IP=$(kubectl get nodes -o jsonpath="{.items[0].status.addresses[?(@.type=='InternalIP')].address}")

3. Access the application at: http://$NODE_IP:$NODE_PORT
